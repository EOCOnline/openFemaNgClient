openFemaNgClient is an Angular (aka Ng) & Typescript (thus ubiqitious JavaScript) browser-only client application to provide an example for accessing one (currently) of the OpenFEMA datasets. A working version/demo is (planned for now) at: http://eocsw.org. (No $ to make it a https (secure) site for now.) Initially this is a proof of concept, with encouragement it could be fortified into an enterprise ready tool.

eoc.online™ provides open-source tools for Emergency Operations Centers and local CERT/VOAD/Citizen Corps groups. For more information check out https://eoc.online - Open Source Tools for EOCs.

We'd love to get your reports of use and suggestions for enhancement. To report issues please visit https://github.com/EOCOnline/OpenFemaNgClient/issues/
